// turbo-all

Этот воркфлоу предназначен для проверки новых механик и существующего функционала проекта через pytest.

### Шаги:

1. **Установка pytest**: Убедитесь, что pytest установлен.
   ```bash
   pip install pytest pytest-asyncio
   ```

2. **Создание директории тестов**: Если папка `tests/` отсутствует, создайте её.
   ```bash
   mkdir -p tests
   ```

3. **Подготовка тестового файла**: Убедитесь, что для тестируемой механики существует файл `tests/test_<name>.py`. Если его нет, создайте базовый тест, импортирующий нужные модули из `src/`.

4. **Запуск тестов**: Выполните pytest для запуска тестов. Используйте флаг -v для детального вывода.
   ```bash
   pytest -v tests/
   ```

5. **Анализ результатов**:
   - Если тесты прошли успешно, зафиксируйте результат.
   - Если тесты упали, проанализируйте Traceback, исправьте код в `src/` или тесте и запустите шаг 4 повторно.

// turbo-all

Этот воркфлоу выполняет полную очистку окружения и пересобирает проект, чтобы убедиться, что все изменения кода и зависимостей применились корректно.

### Шаги:

1. **Остановка окружения**: Полностью останавливаем и удаляем текущие контейнеры и сети.
   ```bash
   docker compose down
   ```

2. **Пересборка и запуск**: Собираем образы заново (без кэша, если нужно, но обычно --build достаточно) и запускаем в фоновом режиме.
   ```bash
   docker compose up --build -d
   ```

3. **Проверка**: Выводим статус запущенных контейнеров.
   ```bash
   docker compose ps
   ```

# Discord Wizard Bot 🪄 (DeepSeek Edition)

Профессиональный инструмент для автоматической настройки Discord серверов с помощью ИИ DeepSeek v4 Flash через Timeweb Cloud AI.

## Новые возможности (v2.6)
- 🏗 **Агентная архитектура**: Специализированные ИИ-агенты для инфраструктуры, пользователей и контента.
- 🚀 **Пайплайны и Реактивность**: Сложные цепочки задач с передачей данных (например, ID созданной категории передается в создание канала).
- 🎨 **Эстетика и Эмбеды**: Создание профессиональных информационных сообщений с использованием вебхуков.
- 🌐 **Web Researcher**: Агенты могут искать информацию в интернете для контекста.

## Структура проекта
```text
WizardBot/
├── src/
│   ├── ai/          # Обработка DeepSeek (Timeweb)
│   ├── bot/         # Логика Discord бота
│   ├── core/        # Ядро системы (биллинг, конфиги)
│   └── tools/       # Инструменты для взаимодействия с Discord API
├── main.py          # Точка входа
└── .env             # Конфигурация
```

## Установка

1. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
2. Настройте `.env`:
   - `DISCORD_TOKEN`: Токен бота.
   - `TIMEWEB_API_KEY`: API ключ от Timeweb Cloud.
   - `ADMIN_ID`: Ваш Discord ID.
3. Запустите:
   ```bash
   python main.py
   ```

## Примеры промптов
- `!wizard Создай структуру сервера для гильдии в MMO: категории 'ИНФО', 'ОБЩЕНИЕ', 'РЕЙДЫ' с соответствующими каналами.`
- `!wizard Найди в сети правила игры WoW и напиши их в канале #rules в красивом эмбеде.`
- `!wizard Переименуй всех пользователей на сервере, добавив префикс [Member].`

## Технические детали
Проект использует **Timeweb Cloud AI** для работы с DeepSeek. Все асинхронные операции оптимизированы для Discord.py.

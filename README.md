# Discord Wizard Bot 🪄 (Structured Version)

Профессиональный инструмент для автоматической настройки Discord серверов с помощью ИИ Gemini.

## Новые возможности (v2.0)
- 🏗 **Структурированный проект**: Разделение на логику чата, инструменты и ядро бота.
- 🎨 **Эстетика и Эмбеды**: Бот умеет создавать красивые информационные сообщения с заголовками, картинками и цветами.
- 🔗 **Анализ примеров**: Вы можете скинуть ссылку на сообщение с другого сервера (доступного боту), и он проанализирует его стиль и контент для повторения.
- 🎭 **Вебхуки**: Поддержка отправки сообщений от любого имени и с любым аватаром (как это делают профессиональные конфигураторы).

## Структура проекта
```text
WizardBot/
├── src/
│   ├── ai/          # Обработка Gemini
│   ├── core/        # Ядро Discord и конфиги
│   └── tools/       # Описание функций для ИИ
├── main.py          # Точка входа
└── .env             # Ваши ключи
```

## Установка

1. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
2. Настройте `.env`:
   - `DISCORD_TOKEN`
   - `GEMINI_API_KEY`
3. Запустите:
   ```bash
   python main.py
   ```

## Примеры промптов
- `!wizard Создай канал #rules и напиши там правила сервера в красивом эмбеде. Используй синий цвет и добавь пункты про уважение и спам.`
- `!wizard Вот пример сообщения: https://discord.com/channels/.../... Изучи его и сделай похожее информационное сообщение в канале #info, но от имени 'Старейшина' с иконкой мага.`
- `!wizard Полностью настрой сервер под уютное сообщество художников.`

## Важно
Для корректной работы вебхуков и эмбедов убедитесь, что у бота есть права на **управление вебхуками** и **управление каналами**.
